🚀 JavaScript Mini Apps Collection

A polished and interactive single-page web application built using HTML, CSS, and JavaScript, showcasing multiple mini projects in one place. This project demonstrates core frontend concepts, DOM manipulation, event handling, and UI design with a professional layout.

📌 Overview

This project is a collection of real-world JavaScript mini applications, all integrated into a single responsive webpage. It is designed to highlight practical frontend development skills and clean UI implementation.

✨ Features
🔹 UI & Experience
Modern and responsive design
Theme toggle (Dark/Light mode)
Live digital clock

🔹 Mini Applications Included
Name Validator
Validates user input with real-time feedback
Character Counter
Displays live character count while typing
Click Counter
Simple counter with interactive button
Random Quote Generator
Generates a new quote on each click
Image Slider
Navigate images with Previous/Next controls
Live Search Filter
Filters list items dynamically
Async Demo
Demonstrates JavaScript asynchronous behavior
Contact Form
Add and display user data in a table
Delete functionality included
Simple Calculator
Perform basic arithmetic operations
To-Do List
Add, complete, and delete tasks

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & layout
JavaScript (ES6+) – Logic & interactivity

📂 Project Structure
project-folder/
